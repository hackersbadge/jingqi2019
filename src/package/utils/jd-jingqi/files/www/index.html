2019京麒国际安全峰会（JD SECURITY SUMMIT，简称京麒安全峰会），是在主管部门的指导下，由京东安全主办，并联合业界和社区共同推出的安全产业和技术峰会，至今已经连续举办了六届。

2019京麒国际安全峰会，将以『捍卫信任』为核心主题，聚焦在互联网企业安全体系建设的视角，以产业互联网背景下的新技术、新风险的应对策略和方案，生态安全合作和创新共享，前瞻安全技术创新和安全人才建设与培养，为主要话题方向，邀请来自国内外的知名安全专家和企业安全高管，分享安全智慧，传递极客精神，为中国互联网安全建设提供一个世界级的交流研讨平台。

本届京麒安全峰会，将携手OWASP、看雪等国内外战略合作伙伴和安全智库，覆盖企业安全体系建设、生态安全、AIoT、数据安全、云安全、威胁情报共享、隐私保护等多个安全领域，打造面向未来网络发展的安全智慧盛宴！

培训课程

京麒国际安全峰会将邀请国际顶尖技术专家举办为期1天的Red Team高级攻击技术与系统化渗透培训，每一位参加培训课程的人都将实际动手操作，这将对技术提升有非常大的帮助。

安全领袖峰会

探讨安全行业趋势的CISO高峰论坛，聚焦在互联网企业安全体系建设的视角，以产业互联网背景下的新技术、新风险的应对策略和方案，生态安全合作和创新共享，前瞻安全技术创新和安全人才建设与培养，为主要话题方向，邀请来自国内外的知名安全专家和企业安全高管，分享安全智慧，传递极客精神，为中国互联网安全建设提供一个世界级的交流研讨平台。

安全技术峰会

一场纯粹的技术盛宴，会议探讨5G、AI、IoT、区块链等前沿领域的安全问题，展示最新研究成果，届时来自checkr1an战队的ios安全专家、国际顶尖黑客、佐治亚大学、南洋理工大学教授将共同用技术“捍卫信任”。

OWASP企业安全论坛

众所周知，OWASP除了耳熟能详的TOP10，作为一个开源的、非盈利的全球性安全组织，致力于应用软件的安全研究，向社会和各行业、组织贡献了大量的安全标准、安全测试工具、安全指导手册等。OWASP企业安全专题论坛将以互联网甲方企业安全视角聚焦，在当前安全风险和合规背景下，分享企业安全运营和QWASP最新开源项目，为甲方安全的建设和运营奏响开源、建设、运营的交响曲。

京麒白帽盛典

京麒白帽盛典也是火石计划的年终收官活动，来自京东、腾讯、唯品会的安全专家，共同为观众呈现一场集技术、趣味、娱乐为一身的白帽盛典。

互联网安全城市巡回赛北京站

由京东安全联合i春秋共同举办的互联网安全城市巡回赛，汇聚10支顶尖白帽团队，在现场深度挖掘6家SRC漏洞。全国TOP50白帽子齐聚京城，3小时的巅峰极客对决，年度最值得期待的赛事，豪华的白帽子阵容，尽在2019京麒国际安全峰会！

极客Village

打破常规、拒绝套路，这里为大开脑洞的极客们准备了各种酷炫的智能设备，你可以拿起电烙铁制作自己的专属硬件，尽情探索Badge中的神秘功能，甚至操作机器人在墙壁上作画。
